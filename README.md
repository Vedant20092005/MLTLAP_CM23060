# 🚀 Machine Learning Tool Lab

## 📌 About the Repository

This repository contains all the practical assignments and experiments completed as part of the **Machine Learning Tool Lab** course.

It showcases a variety of machine learning and deep learning applications implemented using **Python** and **TensorFlow.js**, with a focus on real-world problem solving and model deployment.

---

## 🎯 Objectives

* To understand the core concepts of Machine Learning
* To implement practical machine learning models
* To explore browser-based and real-time applications
* To gain hands-on experience with modern ML tools

---

## 🛠️ Technologies Used

* Python
* TensorFlow.js
* NumPy and Pandas
* Matplotlib
* HTML, CSS, JavaScript
