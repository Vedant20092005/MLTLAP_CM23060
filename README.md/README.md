Machine Learning Tools Lab (MLTLAP_CM23060)

Welcome to my Machine Learning Tools Lab repository.

This repository contains browser-based Machine Learning practicals implemented using TensorFlow.js, focusing on building, training, evaluating, and deploying models directly in a web environment.

The objective of this lab is to provide hands-on experience with modern ML techniques while understanding how models can run efficiently inside a browser.

👨‍🎓 About
Course: Machine Learning Tools Lab (CM23060)
Repository: MLTLAP_CM23060
Technology Stack: JavaScript + TensorFlow.js
Platform: Web Browser
📁 Repository Structure

Each folder in this repository represents a separate practical implementation and contains:

Source code (index.html or related JS files)
Output images (if applicable)
Result documents (PDF, screenshots)

All experiments are committed separately to maintain clear version history.

Practical Experiments in this Repo:

Linear Regression with Synthetic Data – Learn linear regression and plotting predictions.
Text Sentiment Analysis – Browser-based sentiment classifier for text input.
Digit Recognition – Recognize handwritten digits using a pre-trained model.
Image Classification with Pre-trained Model – Classify uploaded images directly in the browser.
Webcam-based Object Detection – Real-time detection using your webcam.
📌 Conclusion

This lab strengthened my understanding of Machine Learning concepts through practical implementation in a browser environment. It bridges the gap between theory and real-world applications by focusing on interactive, real-time ML systems.
